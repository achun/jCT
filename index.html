关于 javascript 模板引擎

JavaScript Common Templates 是一个 javascript 模板引擎,可以运行在浏览器中.最早的托管于 googlecode. javascript 模板引擎其实也很丰富.采用的技术手段也各不相同.jCT采用的是原生 javascript 语法. 类似的也有一些,其中 artTemplate,采用的比较直接.jCT转移到这里,正是看到了 artTemplate 的发布.jCT开发的比较早,在转移到这里之前没有采用字符串拼接的方法,原因是早期的浏览器用字符串拼接效率比较低.artTemplate 的发布让作者发现这几年来现代浏览器字符串拼接的效率早已超过了数组 push 方法.对于采用原生javascript语法的模版引擎来说最终生成的代码在效率上不会有很大差异.不同的引擎会有一些使用上的侧重点.后续会详细介绍
